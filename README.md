# littlekeyboard
My own version of the hackpad from Hack Club

BOM:
6x Switches
1X Seeed Studio Xiao Rp2040
1X 0.91" OLED Screen
1X EC11 Rotary Encoder
2X SK6812 MINI-E RGB LED
1X Case (2 Printed Parts)
4X M3x16mm screws
4X M3x5mx4mm heatset inserts
6X Keycaps
